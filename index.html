Sobre o Projeto

ReciclaJá é uma startup de impacto ambiental que desenvolveu uma plataforma tecnológica para conectar geradores de resíduos recicláveis a coletores especializados, otimizando todo o processo de coleta seletiva através de inteligência de dados.

Problema que Resolvemos

No Brasil, apenas 4% dos resíduos sólidos urbanos são reciclados de forma adequada. A falta de infraestrutura, informação e incentivos econômicos são as principais barreiras para aumentar este índice.

Nossa Solução

Desenvolvemos um ecossistema completo que inclui:

  • API para integração com sistemas municipais e empresas
  • Aplicativo para agendamento de coletas por cidadãos
  • Plataforma de gestão para cooperativas de catadores
  • Sistema de pontuação e benefícios para usuários

Nossa API

A API ReciclaJá fornece todas as funcionalidades necessárias para integrar sistemas de gestão de resíduos, aplicativos móveis e plataformas de cooperativas em um ecossistema unificado.

Autenticação Segura

JWT Token com roles para diferentes tipos de usuários (cidadãos, coletores, administradores).

RESTful Endpoints

Padrão REST com status HTTP semânticos e respostas em JSON.

Webhooks

Notificações em tempo real para atualizações de status de coleta.

Principais Endpoints

POST /api/auth/login

Autenticação de usuários e geração de JWT token.

email: string password: string
GET /api/collections

Lista todas as coletas agendadas (requer autenticação).

status: string date: string limit: number
POST /api/collections/schedule

Agenda uma nova coleta de materiais recicláveis.

materials: Material[] address: Address preferredDate: string
GET /api/materials

Lista todos os materiais recicláveis aceitos pelo sistema.

type: string pointsPerKg: number

Tecnologias Utilizadas

Nossa stack tecnológica foi cuidadosamente selecionada para garantir desempenho, segurança e escalabilidade para a plataforma ReciclaJá.

Node.js

Ambiente de execução JavaScript para construir a API backend de alta performance.

Express.js

Framework web para Node.js que simplifica a criação de endpoints RESTful.

PostgreSQL

Banco de dados relacional robusto para armazenar todos os dados da plataforma.

Prisma ORM

ORM moderno para acesso seguro e produtivo ao banco de dados.

JWT

Autenticação segura com JSON Web Tokens e controle de acesso por roles.

Swagger

Documentação interativa da API para facilitar a integração de desenvolvedores.

Jest

Framework de testes para garantir a qualidade e confiabilidade do código.

Docker

Containerização para deploy consistente em qualquer ambiente.

Exemplo de Uso

// Exemplo de requisição para agendar coleta
const response = await fetch('https://api.reciclaja.com/v1/collections/schedule', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${token}`
  },
  body: JSON.stringify({
    materials: [
      { type: "plastic", quantity: 2.5 },
      { type: "paper", quantity: 1.2 }
    ],
    address: {
      street: "Rua das Flores",
      number: "123",
      city: "São Paulo"
    },
    preferredDate: "2023-11-15"
  })
});

// Resposta da API
const data = await response.json();
console.log(data);
